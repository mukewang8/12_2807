<h3>课程介绍：</h3>
<p>患有数学恐惧症？对希腊字母感到无所适从？本课程将用一点点的数学知识，教会你如何作为统计学原理的消费者，将统计思维运用到浩瀚的社会科学研究中。我们相信，逻辑思维的训练比计算能力更重要。课程结束，你将对如何提炼纷繁数据中的有效信息有更好的感觉；别人研究中的统计分析将不再是你的阅读禁区；你将学会如何作为统计学原理的消费者，将统计思维应用到自己的研究中去。</p>
